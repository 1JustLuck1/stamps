3.3. Угол, шаг, ширина

Формула ширины полосы (ленты):\[B_п = [{A + 2(a+δ)+z_н+δ'}]_{-δ},\]

При самых неблагоприятных условиях штамповки, формула гарантирует сохранение наименьшего значения перемычки a.

Если штамповка осуществляется с применением шаговых ножен и расстояние между ножами равно ширине штампуемой детали, то ширину полосы также рассчитывают по формуле. Если же после обрезки полосы шаговыми ножами остается еще боковая перемычка a, то при штамповке с одним шаговым ножом ширину полосы рассчитывают по формуле: \[B_п = [{A + 3a+2δ)+z_н+δ'+δ_{нж}}]_{-δ},\]

А при штамповке с двумя шаговыми ножами: \[B_п = [{A + 2(2a+δ)+z_н+δ'+δ_{нж}}]_{-δ},\]

Гарантированный наименьший зазор Zн между направляющими планками и полосой и допуск δ' на расстояние между направляющими планками

По рассчитанной ширине полосы определяют число полос, получаемых из листа с заданными размерами, а по длине полосы и шагу t вырубки - число заготовок, получаемых из полосы. После этого вычисляют общее число заготовок, получаемых из листа, норму расхода материала и коэффициент его использования
